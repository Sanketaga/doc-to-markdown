# doc-to-markdown

Convert PDF, DOCX, PPTX, HTML, TXT, and images to clean, AI-optimized Markdown — entirely in your browser. No uploads, no servers, 100% private.
➡️ **[Live Demo](https://sanketaga.github.io/doc-to-markdown/)**
## Features
- Drag-and-drop or paste files — instant conversion
- Handles PDF, DOCX, PPTX, HTML, TXT, PNG, JPG, WEBP
- OCR for scanned PDFs and handwritten text
- Removes page numbers, headers, footers, duplicates
- Preserves headings, lists, tables, and code blocks
- Dark mode support
- Copy to clipboard or download `.md` file
- Everything runs locally — zero server costs
## Usage
Open the [live site](https://sanketaga.github.io/doc-to-markdown/), drop a file, and download the Markdown.
## Built With
- PDF.js · Mammoth · Turndown · Tesseract.js · JSZip
